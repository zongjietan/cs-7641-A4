## Requirements
* Use python version later than 3.5
* Setup a python virtual environment by 
  Run python 3 -m vena env
  Run source env/bin/activate
* Run 'pip install -r requirements.txt'

The 5x5 and 20x20 Frozen Lake Grid World are generated already in the Data folder
* In the `src` folder run the following code:
  * Run python policy_iteration.py 
  * Run python value_iteration.py
  * Run python q_learning.py
